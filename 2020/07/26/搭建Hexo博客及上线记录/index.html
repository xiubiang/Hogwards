搭建Hexo博客及上线记录

记录搭建Hexo博客流程

人在做决定的时候 需要很长的时间 反复和自己争辩 认定自己百分百是对的 但多数时候人是不经过大脑的 只是内心有个声音 你这么做是对的 所以我在

搭建Hexo博客需要确认四类: Hexo的node包，node环境，git工具，github账号；

ready：

1.下载node包：
下载地址 _下载稳定版即可

2.申请github账号：
此处不详细介绍过程

3.下载git工具:
因为被墙，需要通过F12找到对应链接,并使用IDM或者迅雷第三方下载工具下载下载地址

4.下载hexo：
hexo官网 或npm install -g hexo-cli
ps:下载前谨记配置git的SSH，不然会下载失败

construct:

1.下载您需要喜欢的主题包：
下载地址

2.使用主题包：
下载好后放在根目录的themes文件夹中，并把主题名称填写在hexo项目根目录的 _config.yml 的 theme配置属性中
ps:记得在theme的引号后加空格再写主题名称，不用加引号

3.新建github的repository，在头像旁的加号处找到 new repository

4.测试一下是否正常：
使用 hexo server 命令，在浏览器中输入 localhost:4000 即可本地查看

5.开始上线：

运行 npm install hexo-deployer-git –save 命令

把之前新建的repository地址先done下来

把项目复制进done下来的文件夹中

再运行 hexo g d

这时去仓库查看是否上传成功，并新建文件CNAME名称的文件，没有文件后缀，内容是你后期需要域名解析的地址，可以先空着，以备后需

配置setting选项：

6.上线成功：后续上线只需要hexo g d即可重新配置上线
